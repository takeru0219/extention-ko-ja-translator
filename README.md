# Extension KO-JA Translator

A browser extension that translates Korean text to Japanese.

## Features

- Instant Korean to Japanese translation
- Simple and intuitive interface
- Quick toolbar access

## Installation

1. Clone this repository
2. Open your browser's extension settings
3. Enable "Developer mode"
4. Click "Load unpacked" and select this directory

## Usage

Click the extension icon and select text to translate from Korean to Japanese.
